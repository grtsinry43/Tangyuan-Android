# Theme Assets Layout

All theme resources are isolated by theme directory.

## Directory convention

- `default/`
- `spring_festival/`
- future themes: `mid_autumn/`, `national_day/`, etc.

Each theme directory must include:

- `manifest.json`
- `stickers/`
- `backgrounds/`
- optional: `fonts/`, `sounds/`

Do not mix files across theme directories.
